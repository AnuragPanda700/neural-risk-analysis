ğŸš€ Installation & Setup
To run this project locally, you must start both the backend and the frontend servers.

1. Prerequisites
Node.js installed on your machine.

A Google Gemini API Key (Obtainable from Google AI Studio).

2. Backend Setup (The Brain)
Bash
cd sentinel-backend
npm install
Create a .env file in the sentinel-backend folder.

Add your API key: GEMINI_API_KEY=your_actual_key_here

Start the server:

Bash
node index.js
(Server runs on http://localhost:5001 by default)

3. Frontend Setup (The Dashboard)
Open a new terminal:

Bash
cd financial-risk-analysis
npm install
npm run dev
(Open the provided Localhost link in your browser)
